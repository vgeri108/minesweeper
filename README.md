## Telepítés
Az Aknakereső telepítője letölthető [innen](inno-setup/scripts/Output/minesweeper_setup.exe).

## Irányítás
- Kurzor mozgatás: `Nyílgombok`  
- Mező kiásása: `W`  
- Mező bezászlózása: `Space`  
- Kilépés a menübe: `Escape`

Beta 1.6.3 -tól már álítható a menüben.
